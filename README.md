# catppuccin matcha

a cozy, matcha-infused theme for the mindful spirit! this is a custom, nature-inspired spin on the catppuccin palette, blending deep forest tones with soft cream highlights and rich green accents.

---

## 🎨 colors

* **base**: `#1a2418` (dark forest background)
* **text**: `#d8ead0` (soft matcha foam)
* **accent / green**: `#a6d189` (ceremonial matcha)
* **rosewater**: `#e8f4d2` (sweet cream)
